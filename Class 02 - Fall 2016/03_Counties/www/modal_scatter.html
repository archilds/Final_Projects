On this page you can create scatter plots of different variables. You can choose your x and your y axis, and then select a theme and change the formatting options. Below the chart you can optionally add a simple linear regression line of the x axis variable on the y axis variable, or fit a LOESS curve on the scatterplot.

Try this:

  1. X-axis: Percent Trump, Y-axis: % of County, Born Outside U.S. - add a LOESS Curve
  2. X-axis: Percent Trump, Y-axis: % of County, Exercise - add a Regression Line
  3. X-axis: Percent Obese, Y-axis: Hospital Mortality Rate - add a LOESS Curve
