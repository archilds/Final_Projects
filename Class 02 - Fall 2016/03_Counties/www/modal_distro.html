On this page you can begin to visualize the data by creating histograms or bar plots of the variables that have been uploaded. First, select if you would like a histogram or bar chart. Next, you can select what variable you would like to visualize. If you are creating a histogram, you can also choose the bin size to change the appearence of the chart. Depending on the type of variable you have selected, an appropriate table of summary statistics will also be created. You can also change the general format and colors of the charts to be presented.
